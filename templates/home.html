Sports Recommendation System for Cricket

Recommendation Logic 1: Normal Recommendation

Recommendation Engine for Cricket or R.E.F.C is a web platform that uses two different recommendation logics to recommend a team of 11 players most suitable for the next match. Recommendation Engines are applications or softwares that assess the existing / past data and provide a recommendation or suggestion for the future choices. Type of pitch, type of weather is asked to the user which is used by this method to recommend the players by using a simple logic derived using pure performance statistics.

Recommendation Logic 2: K-means Recommendation

This is the second method that the user can use for recommendation of players. Like first method that is mentioned here, this method also takes choices of type of pitch and type of weather and provides recommendation, but this method uses a mathematical model called k-means clustering to calculate groups of candidates by categories or other performance factors and then provides recommendation to the user. Both the methods are efficient but the usability is user dependent. The user can select type of recommendation method while choosing the match conditions.
